Задание по курсу онлайн-школы Thinknetica

www.thinknetica.com

Приложение для создания и выполнения разных тестов. Есть две группы: пользователи (могут выполнять тесты), администраторы (создают тесты)

Возможности TestGuru: Создание и управление тестами, аутентификация, базовое разграничение прав, выполнение тестов, публикация результатов в соцсети
